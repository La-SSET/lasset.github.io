# LaSSET

## 🔧 Getting Started

### **1️⃣ Clone the Repository**

```sh
git clone https://github.com/La-SSET/LaSSET_FS.git
cd LaSSET_FS
```

### **2️⃣ Install Dependencies**

```sh
npm install
```

### **3️⃣ Run the Development Server**

```sh
npm run dev
```

This starts the project with **Hot Module Reloading (HMR)** enabled.
